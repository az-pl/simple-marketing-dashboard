## Simple marketing A/B dashboard 
It allows an executive to easily track measures during an ongoing a/b testing

Based on a dataset from [kaggle](https://www.kaggle.com/datasets/amirmotefaker/ab-testing-dataset)

### Demo - https://www.youtube.com/watch?v=ki8oXEjkE5w

It presents key metrics across:
- impressions
- click-through rate
- cart-to-purchase rate
- total spend and purchases
- post per purchase and purchase conversion rate

Two graphs:
- funnel analisis
- webclicks by date

Filter by date allows to select a desired dateframe.

Tools: power bi, dax measures, modeling, power query, m language, modeling
